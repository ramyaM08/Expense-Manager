# Expense Manager

A simple Python expense tracking application built using Object-Oriented Programming (OOP).

## Features

* Add expenses
* List expenses
* Search expenses by amount
* Search expenses by category
* Search expenses by description
* Delete expenses by amount
* Delete expenses by category
* Delete expenses by description
* Input validation
* Exception handling
* Save expenses to a file
* Load expenses from a file

## Technologies

* Python
* OOP
* File Handling
* Git
* GitHub

## How to Run

python expense_tracker.py

## Project Structure

* Expense class
* ExpenseManager class
* expenses.txt for storing expense records
